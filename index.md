# ReBit

ReBit is an iOS audio player for language learning that helps you repeat the exact parts you need and review them with AI-generated transcripts and learning records.

Import your own audio, split it automatically into short repeatable chunks, and listen to difficult sections as many times as needed. ReBit is designed for shadowing, listening practice, pronunciation training, and reviewing lectures, news, conversations, and podcasts.

## Key Features

- Automatic splitting into short repeatable chunks
- Infinite chunk repeat and custom A–B repeat
- Adjustable pause between repeats for shadowing
- Playback speed from 0.5× to 2×
- On-device AI transcript generation for each chunk
- Edit, copy, and export transcripts as SRT or VTT
- Notes, favorites, hashtags, and search
- Folders for organizing the library
- Repetition counts, streaks, and learning records
- Optional iCloud sync across devices using the same Apple Account

AI speech recognition runs on the device after the selected model has been downloaded. ReBit does not send imported audio to a ReBit-operated server for transcription.

ReBit is free and may display advertisements. ReBit does not impose a numerical limit on the number of audio files you can import, although available device and iCloud storage still apply.

# ReBit

ReBit은 원하는 구간을 반복해서 듣고, AI 스크립트와 학습 기록으로 복습하는 외국어 학습용 오디오 플레이어입니다.

안 들리던 한 문장을, 들릴 때까지. 직접 가져온 오디오를 반복하기 좋은 짧은 구간(청크)으로 자동으로 나누고, 어려운 부분을 필요한 만큼 반복해서 들을 수 있습니다. 영어 셰도잉, 듣기 연습, 발음 훈련, 강의·뉴스·회화·팟캐스트 복습처럼 같은 부분을 여러 번 들어야 하는 학습에 맞췄습니다.

## 주요 기능

- 반복하기 좋은 짧은 청크로 자동 분할
- 청크 무한 반복과 직접 지정하는 A-B 구간 반복
- 셰도잉을 위한 반복 사이 대기시간 설정
- 0.5~2배속 재생
- 기기 내 AI 음성 인식을 이용한 청크별 스크립트 생성
- 스크립트 수정·복사 및 SRT·VTT 자막 내보내기
- 메모, 즐겨찾기, 해시태그와 검색
- 폴더를 이용한 라이브러리 정리
- 반복 횟수, 연속 학습과 학습 기록 확인
- 같은 Apple 계정의 기기 간 선택 가능한 iCloud 동기화

AI 음성 인식은 선택한 모델을 다운로드한 뒤 기기 안에서 처리됩니다. ReBit은 스크립트 생성을 위해 가져온 오디오를 ReBit이 운영하는 별도 서버로 전송하지 않습니다.

ReBit은 무료로 제공되며 광고가 표시될 수 있습니다. 앱이 가져올 수 있는 음원 개수에 별도의 수량 제한을 두지는 않지만, 기기와 iCloud의 사용 가능한 저장 공간에 따라 실제 저장 가능 용량은 달라질 수 있습니다.
