# ReBit Pages

This repository contains the public website files for the ReBit iOS app.

## Pages

- Home: `/`
- Privacy Policy: `/privacy/`
- Support: `/support/`
- Terms of Use: `/terms/`

The published pages describe the current free, advertisement-supported release, on-device AI transcript generation, and optional iCloud synchronization. In-app purchase and five-file-limit text has been removed because those features are disabled in the current release build.

## GitHub Pages Setup

1. Upload the files in this folder to the `main` branch of the public `rebit-pages` repository.
2. Go to repository `Settings` → `Pages`.
3. Set the source to `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save and wait for deployment to complete.
