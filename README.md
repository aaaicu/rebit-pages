# ReBit Pages

This repository contains the public website files for the ReBit iOS app.

## Pages

- Home: `/`
- Privacy Policy: `/privacy/`
- Support: `/support/`
- Terms of Use: `/terms/`

## GitHub Pages Setup

1. Create a public GitHub repository named `rebit-pages`.
2. Upload all files from this folder to the repository.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Set the source to `Deploy from a branch`.
6. Select branch `main` and folder `/root`.
7. Save.
8. After deployment, GitHub will show the published site URL in the Pages settings.

## App Store Connect URLs

Use the published GitHub Pages URLs as follows:

- Marketing URL: the home page
- Privacy Policy URL: the `/privacy/` page
- Support URL: the `/support/` page
